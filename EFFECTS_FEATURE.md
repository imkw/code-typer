# ✨ Code Typer 特效功能

## 新增功能：输入特效

Code Typer 现在支持在字符输入后显示精美的特效！每当输入一个字符时，都会产生令人惊艳的视觉效果。

### 🎆 特效内容

#### 1. 粒子特效
- **触发时机**: 每个字符输入后立即触发
- **持续时间**: 30毫秒
- **视觉效果**: 多彩粒子从输入位置散发
- **粒子数量**: 3-5个随机粒子
- **颜色**: 金色、红色、青色、蓝色、绿色、黄色等多种颜色随机
- **物理效果**: 粒子带有重力效果，会自然下落

#### 2. 状态栏闪烁效果
- **触发时机**: 与粒子特效同时触发
- **持续时间**: 30毫秒
- **视觉效果**: 状态栏右侧快速闪烁特效图标
- **闪烁序列**: ✨ → 💥 → ⚡ → 🎆 → ✨
- **闪烁频率**: 每6毫秒切换一次图标

### 🎛️ 控制方式

#### 状态栏控制
- **位置**: VS Code 状态栏左侧
- **图标**: 
  - 特效开启时: `$(sparkle) 特效开`
  - 特效关闭时: `$(circle-outline) 特效关`
- **操作**: 点击即可切换开启/关闭状态

#### 命令面板控制
- **命令**: `Code Typer: Toggle Effects`
- **快捷操作**: 通过命令面板搜索 "Toggle Effects" 执行

### 📋 使用说明

1. **开启特效**:
   - 点击状态栏的特效按钮
   - 或使用命令面板执行 "Toggle Effects"
   - 看到状态栏显示 "特效开" 即表示已启用

2. **开始打字**:
   - 使用任意 Code Typer 命令开始输入
   - 每输入一个字符都会看到特效

3. **关闭特效**:
   - 再次点击状态栏特效按钮
   - 状态栏显示 "特效关" 即表示已禁用

### 🔧 技术实现

#### 粒子系统
```typescript
interface ParticleEffect {
    id: string;        // 粒子唯一ID
    x: number;         // X坐标位置
    y: number;         // Y坐标位置
    vx: number;        // X方向速度
    vy: number;        // Y方向速度
    life: number;      // 剩余生命周期
    maxLife: number;   // 最大生命周期
    color: string;     // 粒子颜色
}
```

#### 动画循环
- **频率**: 约60fps (每16ms更新一次)
- **物理**: 重力效果 (vy += 0.1)
- **生命周期**: 30毫秒后自动清理

#### 状态栏闪烁
- **实现**: 创建临时状态栏项目
- **自动清理**: 特效结束后自动销毁
- **性能优化**: 使用 try-catch 防止错误

### 🎯 使用场景

1. **代码演示**: 在演示或教学时增加视觉吸引力
2. **直播编程**: 让观众更容易注意到代码输入过程
3. **录制视频**: 为编程教程添加精美的视觉效果
4. **个人偏好**: 纯粹为了享受打字的乐趣

### ⚡ 性能说明

- **轻量级**: 特效系统非常轻量，不会影响 VS Code 性能
- **可控制**: 随时可以关闭特效以节省资源
- **自动清理**: 粒子和临时元素会自动清理，无内存泄漏
- **智能优化**: 没有活跃粒子时自动停止动画循环
